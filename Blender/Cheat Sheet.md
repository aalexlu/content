# Cheat Sheet (Mac)

## General

Views

| Action | Command |
| ------ | ------- |
| front  | 1       |
| back   | **⌘** 1 |
| side   | 3       |
| top    | 7       |

Navigation

| Action | Command   |
| ------ | --------- |
| Pan    | **⇧** MMB |
| Rotate | MMB       |
| Zoom   | MMB wheel |

Transform

| Action    | Command         |
| --------- | --------------- |
| grab      | G               |
| scale     | S               |
| rotate    | R               |
| duplicate | **⇧** D         |
| reset     | **⌥** [G, S, R] |
| delete    | X               |

Other

| Action             | Command |
| ------------------ | ------- |
| edit / object mode | tab     |
| all modes          | ⌃ tab   |
| toggle sidebar     | N       |
| quick favorites    | Q       |
| hide               | H       |
| unhide all         | **⌥** H |
